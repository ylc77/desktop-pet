- 必须使用 beta.3 候选重新执行安装、启动、单实例、托盘、开机启动、正常退出和卸载。

## 公开测试风险

- Windows Authenticode 尚未配置。Tauri Updater 签名只保护更新包完整性和发布授权，不能替代可信 Windows 发布者证书；SmartScreen 仍可能警告或显示未知发布者。
- 真实混合 DPI 双屏、显示器热插拔、睡眠/唤醒、Windows 重启和远程桌面尚未完整执行。
- 尚无当前候选的 8 小时稳定性结果；15 分钟真实性能测试也必须在最终 Release 二进制上重新执行。
- WebView2 缺失时，当前 `downloadBootstrapper` 安装策略需要联网，尚不支持完整离线 Runtime 安装。
- 当前只验证 Windows x64。ARM64、Microsoft Store、macOS 和 Linux 不在本候选范围内。
- 当前素材仍为中性占位角色，不是正式桌宠外观。
