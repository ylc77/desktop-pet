# 0.1.0 公开测试版候选说明（草案）

这是一份候选说明草案，不代表 `0.1.0-beta.1` 已发布。当前真实应用版本仍为 `0.1.0`；生产 updater 公钥/endpoint、两个真实版本升级和多项 Windows Gate 尚未完成，没有生成公开测试版目录或正式 beta 标签。

七酱桌宠是离线优先、角色资源包驱动的 Windows 桌宠应用。当前包含透明置顶窗口、PNG 动画状态机、拖动和点击交互、设置持久化、托盘、单实例、开机启动、全屏隐藏、日志和资源校验。当前画面仍为带 `DEV` 标识的中性几何占位角色，不是正式桌宠外观。

软件品牌、`desktop_pet.exe` 主程序命名和白猫应用图标已经接入；角色包协议仍为 `schemaVersion: 1`，没有制作正式桌宠角色。本候选还加入关于页面、手动检查更新、自动检查开关、脱敏诊断导出和恢复默认设置，以及 Tauri 2 Updater/Process 的安全集成基础。

当前不包含 AI、账号、支付、云数据库、遥测、广告或在线诊断上传。更新服务仍为 `NOT_CONFIGURED`：普通构建不联网，不能下载或安装更新。自动更新发布工具已经能用外部公钥对 artifact 与 `.sig` 做真实密码学验签，并校验 `latest.json` 的精确版本、URL 文件名和实际字节数；但生产私钥未生成、HTTPS endpoint 未选择，也没有完成真实端到端升级。

## 当前限制

- 仅面向 Windows 10/11 x64；真实干净系统验收尚未完成。
- 安装包未签名，可能触发 SmartScreen，发布者可能显示为未知。
- Tauri Updater 签名与 Windows 代码签名是两套机制；前者的基础接入不能消除当前 `NotSigned` 风险。
- 生产 updater 密钥、至少两份安全备份和正式 HTTPS 托管尚未完成。
- 版本 A → B 的真实发现、下载、签名验证、被动安装、重启和数据保留测试尚未执行。
- WebView2 缺失时需要联网下载；完全离线安装尚不支持。
- 当前候选仍需重新执行真实安装、启动、正常退出和卸载；历史 QA 不能作为本候选通过证据。
- Windows 10/11 VM、WebView2 缺失、升级、真实双屏、睡眠/重启和 8 小时测试尚未完成。

项目当前没有公开客服或自动反馈服务。诊断包只由用户主动导出到本地，不会自动上传；测试人员应自行审阅脱敏内容，再将 QA 报告、安装包 SHA-256、Windows 版本和复现步骤交给项目维护者。
