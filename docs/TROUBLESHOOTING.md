# 七酱桌宠故障排查

## 无法启动

- 确认 Windows 10/11 x64 和 Microsoft Edge WebView2 Runtime 可用。
- 检查 `%LOCALAPPDATA%\dev.deskpet.framework\logs\` 中的最近日志。
- 不要安装 Node.js 或 Rust；最终用户运行不需要开发工具。

## 窗口不可见

- 再次启动七酱桌宠，以触发单实例窗口恢复。
- 通过系统托盘中的七酱桌宠图标尝试恢复。
- 显示器刚断开或 DPI 改变时，等待窗口恢复到现有工作区。

## 设置损坏

应用会回退到默认设置，并把损坏文件改名为 `settings.corrupt-<时间戳>.json`。不要公开上传包含本机路径或私人内容的设置和日志。

## 安装失败

- 缺少 WebView2 且断网时，当前小体积安装包不能完成完全离线安装。
- 记录安装器退出码、Windows 版本和安装包 SHA-256。
- 不要关闭 Defender、添加排除项或绕过 SmartScreen。

## 卸载后仍有残留

先确认没有 `desktop_pet.exe` 或卸载器进程。保留现场并记录卸载注册表、安装目录和开始菜单状态，不要直接删除注册表来伪造通过结果。已知问题请参阅 `KNOWN_ISSUES.md`。

项目当前没有公开客服或自动反馈渠道。请把最小复现步骤和脱敏后的 QA 报告交给项目维护者。
