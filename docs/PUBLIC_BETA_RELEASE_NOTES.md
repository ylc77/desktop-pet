# 七酱桌宠 0.3.0 正式版发布说明

发布目标：`v0.3.0` 稳定渠道。本文随候选源码维护；只有生产签名资产、本地验证与远端回下载验证全部通过后才发布 GitHub Release。

七酱桌宠是一款离线优先、角色资源包驱动的 Windows 桌宠应用。当前包含透明置顶桌宠窗口、PNG 动画状态机、拖动和点击交互、设置持久化、系统托盘、单实例、开机启动、全屏自动隐藏、日志诊断、自动更新以及离线外观中心。

## 0.3.0 主要内容

- 设置、外观中心和关于界面采用统一的“暖栖 Fluent”桌面设计语言。
- 外观中心支持官方内置角色、本机 `.qipet` 导入、切换、升级和删除。
- 首次随程序内置正式角色“小幽” `0.4.0`，包含 14 个动作和 110 帧透明 PNG；新增左右观察、眨眼微笑和掌心冰雪魔法三组低频待机动作。
- 拖动小幽时按方向显示双脚合拢、明显离地的冰晶漂浮动画；程序不会自动在桌面巡游。
- 桌宠右键菜单改为紧凑的 Windows Fluent 风格，支持屏幕边缘避让、键盘导航、失焦关闭，并修复菜单区域点击穿透。
- 修复设置和外观中心无法通过右上角 X 关闭的问题。
- 桌宠连续移动改为有上限的原生窗口更新，并在移动停止后执行防抖式越界恢复，避免 Windows 消息队列积压导致未响应。
- 设置写入、角色切换、升级、删除和退出流程使用受控事务与失败回退。
- Updater 使用 HTTPS GitHub Releases、生产公钥和版本化签名资产；只有用户点击“立即更新”后才下载和安装。
- NSIS 安装与卸载界面显式使用简体中文；安装后的主程序文件名保持 `desktop_pet.exe`。

角色包协议保持 `schemaVersion: 1`，应用 identifier 保持 `dev.deskpet.framework`。用户设置、日志目录与已安装 `.qipet` 外观包继续使用兼容路径。

## 安装说明

- 支持 Windows 10/11 x64。
- 对外安装包名称为 `七酱桌宠.exe`；GitHub Updater 的传输资产使用 ASCII 版本化文件名。
- 缺少 Microsoft Edge WebView2 Runtime 时，安装器需要联网下载 Runtime。
- 当前未配置 Windows Authenticode，SmartScreen 可能显示未知发布者；请先核对 Release 中的 SHA-256。

## 已知限制

- 小幽已作为首套正式角色内置；中性几何角色仍保留为损坏资源回退方案。
- 不包含 AI、账号、支付、广告、遥测、云数据库或在线诊断上传。
- Windows ARM64、Microsoft Store、macOS 和 Linux 不在本版本范围内。
- 干净 Windows 10/11、WebView2 缺失、混合 DPI 多屏、睡眠/唤醒和长时间稳定性仍需在更多独立环境持续验证。
- 本版本不重复执行真实 A → B 自动更新生命周期；该取舍由项目所有者确认。生产构建仍必须通过 Updater 配置、签名、哈希、Manifest 和远端资产验证，不能据此宣称自动更新已在本候选上重新完成端到端验证。

诊断包只会由用户主动导出到本地，不会自动上传。项目当前没有账号系统或自动反馈服务。
