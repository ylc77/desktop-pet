# 已知问题

适用版本：`0.3.0`。

## 发布前必须确认

- 生产 Tauri Updater 安装包、`.sig`、`latest.json`、Release Manifest 和 `SHA256SUMS.txt` 必须来自同一个干净提交并相互匹配。
- 最终 NSIS 安装包必须完成本地启动、设置/外观中心关闭、中文安装界面和卸载器检查。
- Windows Authenticode 尚未配置。Updater 签名保护更新包完整性，但不能替代可信 Windows 发布者证书；SmartScreen 仍可能警告或显示未知发布者。
- 本版本按项目所有者决定不重复执行真实 A → B 自动更新生命周期。以前版本的成功结果只能作为历史依据，不能描述为 `0.3.0` 的新端到端证据。

## 环境覆盖限制

- WebView2 缺失时，当前 `downloadBootstrapper` 安装策略需要联网，不支持完全离线安装。
- 真实混合 DPI 双屏、显示器热插拔、睡眠/唤醒、Windows 重启和远程桌面尚未完整覆盖。
- 干净 Windows 10、干净 Windows 11、中文用户名、长路径和 8 小时稳定性仍建议继续收集独立证据。
- 当前只支持 Windows x64；ARM64、Microsoft Store、macOS 和 Linux不在本版本范围内。

## 内容限制

- 当前安装包包含正式角色“小幽”和中性几何回退角色；小幽角色包版本为 `0.4.0`，共 14 个动作、110 帧。
- 应用不包含 AI、账号、支付、广告、遥测、云同步或在线诊断上传。
