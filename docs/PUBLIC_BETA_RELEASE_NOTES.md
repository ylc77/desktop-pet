# 0.1.0 公开测试版候选说明（草案）

这是一份候选说明草案，不代表 `0.1.0-beta.1` 已发布。当前 Gate 为 `NOT_READY`，尚未生成公开测试版目录或正式 beta 标签。

Desk Pet Framework 是离线优先、角色资源包驱动的 Windows 桌宠开发框架。当前包含透明置顶窗口、PNG 动画状态机、拖动和点击交互、设置持久化、托盘、单实例、开机启动、全屏隐藏、日志和资源校验。当前画面仍为带 `DEV` 标识的中性几何占位角色，不是正式桌宠外观。

当前不包含 AI、账号、支付、云数据库、遥测、自动更新或在线服务。角色包协议保持 `schemaVersion: 1`。

## 当前限制

- 仅面向 Windows 10/11 x64；真实干净系统验收尚未完成。
- 安装包未签名，可能触发 SmartScreen，发布者可能显示为未知。
- WebView2 缺失时需要联网下载；完全离线安装尚不支持。
- 最新真实卸载测试存在程序目录、卸载记录和开始菜单残留，公开测试前必须修复。
- Windows 10/11 VM、WebView2 缺失、升级、真实双屏、睡眠/重启和 8 小时测试尚未完成。

项目当前没有公开客服或自动反馈服务。测试人员应将 QA 报告、安装包 SHA-256、Windows 版本和复现步骤交给项目维护者。
